# Technical Task

## Requirements
* PHP7
* MongoDB
* Composer

## Install
```
composer install
```

## Run 
```
php bin/console server:run
open http://127.0.0.1:8000 in browser
```

## Commands
* app:populate:neos - populate NEOs

## Documentation
open http://127.0.0.1:8000/api/doc in browser

## Test
```
cp phpunit.xml.dist phpunit.xml
php vendor/bin/phpunit
```
